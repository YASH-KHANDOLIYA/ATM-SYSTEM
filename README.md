# ATM SYSTEM  

## Overview  
This project simulates an **Automated Teller Machine (ATM)**, demonstrating core concepts of software engineering and object-oriented programming. It also includes project estimation and standardization files such as **COCOMO Estimation**, **ISO Standards**, and **Testing documentation**.  

---

## Project Modules  
| File Name | Description |
|------------|-------------|
| `ATM SYSTEM CODE.cpp` | Main ATM program that performs user authentication, balance check, deposit, withdraw, and PIN management. |
| `ATM SYSTEM TESTING.cpp` | Contains the testing and validation logic for the ATM functionalities. |
| `ATM ISO STANDARDS.cpp` | Describes ISO standards followed in software design and implementation. |
| `ATM COCOMO ESTIMATION.cpp` | Includes cost and effort estimation for the project using the **COCOMO Model**. |
| `README.md` | Project documentation. |

---

## Features  
- User Login using Account Number & PIN  
- Deposit, Withdraw, Balance Inquiry  
- PIN Change Option  
- Transaction Record Handling (optional)  
- File Handling for Data Storage  
- Testing, ISO Documentation, and Cost Estimation modules
